# Підключення каналу

Після введення UID та рефкоду бот попросить посилання на ваш Telegram-канал.

## Прийнятні формати

| Формат | Приклад |
|--------|---------|
| Публічний канал | `https://t.me/yourchannel` |
| Приватний канал | `https://t.me/+AbCdEfGhIjK` |
| Через @ | `@yourchannel` |

## Що відбувається після

1. Бот підписується на ваш канал
2. Це потрібно для автоматичної перевірки постів
3. Статус підписки відображається у профілі

## Статуси підписки

| Статус | Значення |
|--------|----------|
| `subscribed` | Бот підписаний на канал ✅ |
| `pending` | Очікує підключення |
| `failed` | Помилка — зверніться до адміна |

## Приватний канал

Для приватних каналів:
1. Додайте бота як **адміністратора** каналу
2. Потім відправте invite-посилання боту

{% hint style="warning" %}
Один канал може бути зареєстрований лише одним партнером.
{% endhint %}
