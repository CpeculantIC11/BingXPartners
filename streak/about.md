# Що таке стрік

**Стрік** — це кількість місяців поспіль, коли ви підтримуєте ранг **Ambassador або вище** (рівень 3+).

## Як працює

```
Кожне 27 число → автоматична виплата → перевірка рангу:

Ранг ≥ Ambassador → стрік +1
Ранг < Ambassador → стрік = 0 (скид)
```

## Приклад

| Місяць | Ранг | Стрік |
|--------|------|-------|
| Квітень | Ambassador | 1 |
| Травень | Opinion Leader | 2 → бонус +20 |
| Червень | BingX Leader | 3 → бонус +50 |
| Липень | Supporter ❌ | 0 (скид) |
| Серпень | Ambassador | 1 |

## Де видно стрік

- **Головне меню** → рядок «🔥 Бонус стріку»
- **Статистика** → рядок «Стрік» з іконками 🔥
- Біля рангу: `Ambassador x3🔥`

{% hint style="warning" %}
Іконка 🔥 біля рангу відображається **тільки для Ambassador і вище**. Якщо ваш ранг Supporter або Promoter — стрік не показується.
{% endhint %}

## Що скидає стрік?

Якщо на момент виплати (27 числа) ваш ранг нижче Ambassador — стрік обнуляється до 0. Наступний стрік доведеться набирати з початку.
