# 💠 BINGO — Партнерська програма BingX

Ласкаво просимо до гайду партнерської програми **BINGO** від BingX!

Цей бот допомагає вам:

- 📝 Подавати пости з реферальними посиланнями
- 🏅 Відстежувати свій ранг і прогрес
- 💰 Накопичувати та виводити нагороди в USDT або Telegram Stars
- 🔥 Отримувати бонуси за стабільну активність (стрік)

## Швидкий старт

1. Напишіть боту `/start`
2. Натисніть **«Підключитися»**
3. Введіть BingX UID, реферальний код і посилання на канал
4. Готово — подавайте пости і отримуйте BINGO 💠!

## Курси та ліміти

| Параметр | Значення |
|----------|----------|
| 10 BINGO 💠 | = 1 USDT |
| 15 BINGO 💠 | = 100 Telegram Stars |
| Мін. вивід | 50 BINGO 💠 |
| Бонус за 1-й пост | 100 BINGO 💠 |
| Ліміт постів | 1 валідний / день |
| Виплата | 27 числа щомісяця |
| Стрік працює від | Ambassador (рівень 3+) |
