# Нарахування нагород

## Коли нараховуються

**27 числа кожного місяця** — автоматична виплата за поточний сезон.

Сума = нагорода рангу + бонус стріку (якщо є).

## Процес виплати

```
27 число → Система перевіряє:
├── Кількість валідних постів
├── Об'єм торгів рефералів
├── Визначає ранг
├── Перевіряє стрік (Ambassador+)
└── Нараховує BINGO на баланс
```

## Незабрані нагороди

Після нарахування в розділі «💠 Мої BINGO» з'явиться кнопка:

**«🎁 Отримати X BINGO»**

Натисніть — і нагорода зарахується на ваш баланс.

## Бонус за перший пост

При першому валідному пості за весь час участі — одразу нараховується **100 BINGO 💠**. Це одноразовий бонус, не прив'язаний до дати 27 числа.
