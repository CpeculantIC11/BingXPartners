# Як працюють сезони

## Що таке сезон

Один сезон = **один календарний місяць**.

- Сезон 1 = Квітень 2026
- Сезон 2 = Травень 2026
- і т.д.

## Що обнуляється щомісяця

| Параметр | Обнуляється? |
|----------|-------------|
| Валідні пости | ✅ Так → з 0 |
| Об'єм торгів | ✅ Так → з 0 |
| Ранг | ✅ Перераховується з нуля |

## Що НЕ обнуляється

| Параметр | Зберігається? |
|----------|--------------|
| Баланс BINGO 💠 | ✅ Так |
| Стрік | ✅ Так (якщо ранг ≥ Ambassador) |
| Реєстрація | ✅ Так |
| Рефкоди | ✅ Так |
| Канал | ✅ Так |

## Хронологія сезону

```
1 число — Початок нового сезону (пости і об'єм з 0)
    ↓
Протягом місяця — Подаєте пости, реферали торгують
    ↓
27 число — Автоматична виплата за досягнутий ранг
    ↓
Стрік: +1 якщо Ambassador+ | 0 якщо нижче
    ↓
Наступний місяць — Новий сезон
```

{% hint style="info" %}
Щоб підтримувати стрік — потрібно **щомісяця** виконувати умови для Ambassador або вище (мін. 1 пост + $50K об'єму рефералів).
{% endhint %}
