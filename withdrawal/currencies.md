# Валюти та курси

## Доступні валюти

### 💵 USDT

- Курс: **10 BINGO = 1 USDT**
- Вивід на: ваш **BingX UID** (з профілю)
- Потрібно: мати BingX акаунт

| BINGO | USDT |
|-------|------|
| 50 | $5 |
| 100 | $10 |
| 200 | $20 |
| 300 | $30 |
| 500 | $50 |

### ⭐ Telegram Stars

- Курс: **15 BINGO = 100 Stars** (~$1.5)
- Вивід на: ваш **Telegram username**
- Потрібно: мати username в налаштуваннях Telegram

| BINGO | Stars |
|-------|-------|
| 50 | ~333 |
| 100 | ~667 |
| 150 | ~1000 |
| 300 | ~2000 |

## Куди відправляються кошти

| Валюта | Реквізити |
|--------|-----------|
| USDT | Автоматично на ваш BingX UID |
| Stars | Автоматично на ваш @username |

{% hint style="info" %}
Для виводу в Stars обов'язково потрібен **username** в Telegram. Якщо його немає — встановіть у налаштуваннях Telegram.
{% endhint %}
