# Реєстрація в боті

## Крок за кроком

1. Відкрийте бот у Telegram і натисніть `/start`
2. Натисніть кнопку **«Підключитися»**
3. Бот запитає 3 речі по черзі:

### 1️⃣ BingX UID

Числовий ідентифікатор вашого акаунту BingX.

> Де знайти: додаток BingX → профіль → UID, або на сайті bingx.com

### 2️⃣ Реферальний код

Ваш персональний код з партнерського кабінету.

> Де знайти: [hi.bingx.com](https://hi.bingx.com) → Управління посиланнями

Код може бути числовим (`12345`) або кастомним (`CryptaBingX`). Можна вставити повне посилання — бот витягне код автоматично.

### 3️⃣ Посилання на Telegram-канал

Прийнятні формати:
- `https://t.me/yourchannel`
- `https://t.me/+invite_hash` (приватний канал)
- `@yourchannel`

Бот підпишеться на канал для перевірки постів.

## Після реєстрації

Ви побачите головне меню з вашим прогресом:

```
🏅 Ранг: Connected
💰 Баланс: 0 BINGO 💠
☘️ Сезон 1
📝 Пости: 0/1
📈 Об'єм: 0
```
