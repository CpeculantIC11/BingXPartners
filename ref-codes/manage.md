# Управління рефкодами

Рефкоди — це ваші персональні коди з партнерського кабінету BingX. Бот підтримує кілька рефкодів одночасно.

## Перегляд рефкодів

**Головне меню → «🔗 Мої реф-коди»**

Ви побачите список всіх ваших кодів з лічильником `X/MAX`.

## Додавання нового коду

1. Натисніть **«➕ Додати код»**
2. Введіть код одним з варіантів:
   - Тільки код: `CryptaBingX` або `12345`
   - Повне посилання — бот витягне код автоматично

{% hint style="info" %}
Кожному коду можна додати **мітку** (наприклад: «Основна», «Для акцій»), або пропустити цей крок.
{% endhint %}

## Видалення коду

В списку рефкодів натисніть **«❌ Видалити»** біля потрібного коду.

## Де взяти рефкод

1. Перейдіть на [hi.bingx.com](https://hi.bingx.com)
2. Розділ **«Управління посиланнями»**
3. Скопіюйте код (числовий або кастомний)

Або: мобільний додаток → **Центр партнера**
