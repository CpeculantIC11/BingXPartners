# Як подати пост

## Крок за кроком

1. Опублікуйте пост з реферальним посиланням BingX у вашому каналі
2. Скопіюйте посилання на пост
3. В боті: **Головне меню → «📝 Додати пост»**
4. Вставте посилання у форматі:

```
https://t.me/yourchannel/123
```

5. Бот автоматично перевірить пост (до 30 секунд)
6. Результат: ✅ прийнято або ⚠️ помилки

## Що має містити пост

- ✅ Реферальне посилання BingX з **одним з ваших рефкодів**
- ✅ Пост у **вашому зареєстрованому каналі**
- ❌ **Без посилань конкурентів** (Bybit, Binance, OKX, тощо)

## Після подачі

Якщо пост прийнято:
- Він зараховується як валідний
- За перший пост за весь час — бонус **100 BINGO 💠**
- Бот зробить повторну перевірку через 1-24 години

{% hint style="warning" %}
Якщо після подачі видалити пост або прибрати рефку — при повторній перевірці статус зміниться на невалідний.
{% endhint %}
