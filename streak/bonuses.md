# Бонуси за стрік

Стрік дає додаткові BINGO 💠 **зверху** до нагороди рангу.

## Таблиця бонусів

| Стрік (місяці поспіль) | Бонус |
|----------------------|-------|
| 1 | 0 (без бонусу) |
| 2 | **+20 BINGO 💠** |
| 3 | **+50 BINGO 💠** |
| 4+ | **+100 BINGO 💠** |

## Як це впливає на нагороду

Бонус стріку **додається** до нагороди рангу:

```
Поточна нагорода = Нагорода рангу + Бонус стріку
```

### Приклад

**Ambassador (стрік 3):**
- Нагорода рангу: 150 BINGO 💠
- Бонус стріку: +50 BINGO 💠
- **Разом: 200 BINGO 💠**

**BingX Leader (стрік 4+):**
- Нагорода рангу: 300 BINGO 💠
- Бонус стріку: +100 BINGO 💠
- **Разом: 400 BINGO 💠**

{% hint style="success" %}
Максимальна можлива нагорода: **400 BINGO 💠** (BingX Leader + стрік 4+) = **40 USDT / місяць**
{% endhint %}

## Де видно бонус

У головному меню рядок:
- `🔥 Бонус стріку: +50 BINGO 💠` — якщо є бонус
- `🔥 Бонус стріку: немає` — якщо стрік < 2 або ранг < Ambassador
