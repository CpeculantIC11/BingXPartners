# Ліміти та правила

## Ліміт постів

| Правило | Значення |
|---------|----------|
| Валідних постів на день | **1** |
| Цільова кількість на сезон | **4** (для макс рангу) |
| Мінімум для нагороди | **1** валідний пост |

## Правила подачі

- Один пост можна подати **тільки один раз**
- Один URL не може бути поданий **різними партнерами**
- Пост має бути у вашому **зареєстрованому каналі**
- Пост повинен містити **ваш рефкод** (не чужий)

## Якщо пост не пройшов перевірку

Перевірте:
1. Чи є в пості **реферальне посилання BingX**
2. Чи використали **свій рефкод** (не чужий)
3. Чи немає в пості **посилань на інші біржі**
4. Чи пост опублікований у **вашому каналі**

Виправте пост і подайте його посилання заново.

{% hint style="info" %}
Невалідний пост зберігається в системі, але **не враховується** при підрахунку рангу та нагороди.
{% endhint %}

## Перегляд поданих постів

**Головне меню → «📋 Мої пости»**

Побачите список всіх постів за поточний сезон зі статусами (✅ / ❌).
